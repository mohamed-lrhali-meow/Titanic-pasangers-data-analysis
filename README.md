# Titanic Data Analysis

This project performs exploratory data analysis (EDA) and visualization on the Titanic dataset.

## What's Included

**Data Cleaning:**
- I loaded the Titanic dataset and inspected its structure
- I handled missing values: filled missing cabin information with "Unknown" and imputed age values using the mean
- I identified and examined data quality issues

**Exploratory Data Analysis:**
- I analyzed passenger demographics including sex distribution and age statistics
- I calculated survival rates and survival statistics by passenger sex
- I explored survival patterns across passenger classes
- I examined age and passenger class distributions

**Data Visualization:**
- I created histograms to visualize age and passenger class distributions among survivors
- I generated bar charts showing survival rates by passenger class (1st, 2nd, 3rd class)

## Key Findings

Through my analysis, I discovered patterns in Titanic survival rates across different passenger demographics and socioeconomic factors. I found clear differences in survival outcomes based on passenger class and gender.
